## About me:

I am a Software Quality Assurance Engineer with eight years of experience spanning hardware, firmware, network & protocol, SaaS, web, and digital retail technology. Over my career, I have gained expertise across a variety of industries, technologies, and SDLC methodologies—from agile-waterfall hybrid startups to international SaaS organizations delivering enterprise software.

As my career has progressed, my focus has shifted from manual testing to developing test automation. My journey into automation began with learning Python to create a script that automated a cumbersome manual testing process. Since then, I have gained professional experience contributing to a BDD Python framework for API and UI (Selenium) testing, a TypeScript/Cypress front-end framework, and a Go back-end testing library.
